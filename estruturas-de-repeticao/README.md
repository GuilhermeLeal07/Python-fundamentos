
# 🔁 Estruturas de Repetição em Python

Este diretório contém exercícios práticos focados no uso de estruturas de repetição na linguagem Python.

## 📚 Sobre o módulo

As estruturas de repetição são fundamentais para automatizar tarefas e controlar fluxos que exigem execução contínua de comandos.

Neste módulo, são explorados os principais conceitos relacionados a loops, com foco na construção de uma base sólida para resolução de problemas.

## 🚀 Conteúdos abordados

* Laço `for`
* Laço `while`
* Validação de entrada com repetição
* Contadores e acumuladores
* Controle de fluxo com condições

## 🧠 Objetivo

Desenvolver a capacidade de:

* Repetir ações de forma controlada
* Validar dados de entrada do usuário
* Criar lógicas mais dinâmicas
* Preparar para construção de sistemas mais complexos

## 🗂️ Estrutura dos exercícios

Os arquivos seguem um padrão de nomeação:

```
exXXX_nome_do_exercicio.py
```

Exemplo:

```
ex001_for_basico.py
ex002_while_validacao.py
```
