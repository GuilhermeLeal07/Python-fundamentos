# 📘 Introdução e Primeiros Comandos

Esta pasta faz parte do repositório **Python-fundamentos**
e contém exercícios iniciais voltados para a base da linguagem Python.

---

## 🎯 Objetivo

Construir uma base sólida em:

* Sintaxe básica
* Uso da função `print()`
* Entrada de dados com `input()`
* Conversão de tipos (`int`, `float`, `str`)
* Operadores aritméticos
* Ordem de precedência
* Cálculos matemáticos simples
* Manipulação de strings
* Estruturas condicionais (`if`, `elif`, `else`)

---

## 📂 Estrutura dos Exercícios

Os arquivos seguem um padrão de organização:

```
ex001-nome-do-exercicio.py
ex002-nome-do-exercicio.py
...
```

Cada arquivo contém:

* 📌 Descrição do exercício
* 🧠 Lógica aplicada
* 💻 Código organizado e comentado

---

## 🚀 Evolução do Código

Ao longo dos exercícios, há evolução gradual na forma de escrever o código:

* Uso inicial de `.format()`
* Transição para **f-strings** (forma moderna)
* Melhor organização e legibilidade
* Padronização de saída no terminal

---

## 🎨 Destaque
