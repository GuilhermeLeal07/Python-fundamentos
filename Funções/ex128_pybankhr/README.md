# 🏦 pyBankHR

Este diretório contém o desenvolvimento de um sistema bancário orientado a objetos criado durante os estudos de Python.

## 📖 Descrição do projeto

O **pyBankHR** simula um pequeno sistema bancário utilizando Programação Orientada a Objetos (POO).

O projeto foi estruturado em múltiplas classes para representar clientes, contas bancárias, contas especiais, contas poupança e extratos, aplicando conceitos fundamentais da orientação a objetos.

## 📚 Sobre o projeto

O sistema demonstra como dividir responsabilidades entre diferentes classes, tornando o código mais organizado, reutilizável e fácil de manter.

Cada classe possui uma responsabilidade específica dentro da aplicação.
