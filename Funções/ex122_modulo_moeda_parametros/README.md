# 💰 Módulo Moeda com Parâmetros

Este exercício expande o módulo de moeda, permitindo que as funções retornem valores formatados opcionalmente.

## 🚀 Conteúdos abordados

- Modularização
- Parâmetros opcionais
- Operador ternário
- Reutilização de funções
- Formatação monetária

## 📂 Arquivos

- moeda.py
- teste.py

## 🧠 Objetivo

Criar funções mais flexíveis, permitindo ao usuário escolher entre resultados numéricos ou formatados.
