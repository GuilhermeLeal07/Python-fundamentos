# 📂 Sistema de Cadastro de Pessoas

Este diretório contém a versão completa do sistema de cadastro de pessoas desenvolvido durante os estudos de Funções, Modularização e Manipulação de Arquivos em Python.

## 📖 Descrição do desafio

**Correspondente ao Desafio 115C do Curso de Python – Gustavo Guanabara.**

Nesta etapa foi concluído o sistema de cadastro de pessoas utilizando módulos, tratamento de exceções e persistência de dados em arquivos texto.

O programa permite cadastrar pessoas, listar todos os registros armazenados e mantém as informações salvas mesmo após o encerramento da aplicação.

## 📚 Sobre o projeto

O sistema foi dividido em módulos independentes, cada um responsável por uma parte da aplicação.

Essa organização facilita a manutenção, reutilização de código e futuras expansões do projeto.

## 🚀 Conteúdos abordados

* Modularização
* Organização em pacotes
* Manipulação de arquivos
* Leitura e escrita em arquivos texto
* Tratamento de exceções
* Validação de entrada de dados
* Estruturação de sistemas em Python

## 🧠 Objetivo

Desenvolver a capacidade de:

* Construir aplicações organizadas em módulos.
* Trabalhar com persistência de dados.
* Criar interfaces reutilizáveis.
* Validar entradas do usuário.
* Produzir códigos mais limpos, reutilizáveis e profissionais.

## 💡 Observações

* O arquivo de cadastro é criado automaticamente caso não exista.
* O sistema utiliza tratamento de exceções para tornar a aplicação mais segura.
* As responsabilidades foram separadas em módulos específicos.
* A interface do usuário foi organizada em funções reutilizáveis.
