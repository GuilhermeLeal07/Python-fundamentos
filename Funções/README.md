# 🧩 Funções em Python

Este diretório contém exercícios práticos focados na criação e utilização de funções na linguagem Python.

## 📚 Sobre o módulo

As funções são estruturas fundamentais para organizar programas, evitar repetição de código e tornar soluções mais reutilizáveis e legíveis.

Neste módulo, são explorados os conceitos essenciais relacionados à criação de funções e à modularização de programas.

## 🚀 Conteúdos abordados

* Criação de funções com `def`
* Parâmetros e argumentos
* Escopo de variáveis
* Retorno de valores com `return`
* Documentação de funções
* Modularização de código

## 🧠 Objetivo

Desenvolver a capacidade de:

* Criar funções reutilizáveis
* Organizar melhor o código
* Evitar repetição de comandos
* Separar responsabilidades dentro de um programa
* Construir soluções mais profissionais e escaláveis

## 🗂️ Estrutura dos exercícios

Os arquivos seguem um padrão de nomeação:

```text
exXXX_nome_do_exercicio.py
```

Exemplo:

```text
ex111_calculo_area_terreno.py
ex112_funcao_escreva.py
```
