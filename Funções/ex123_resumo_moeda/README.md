# 💰 Resumo Monetário

Este diretório contém a evolução do módulo de moeda desenvolvido durante os estudos de Funções em Python.

## 📚 Sobre o módulo

Neste exercício foi criada uma função capaz de gerar um resumo completo de um valor informado, exibindo operações monetárias de forma organizada.

## 🚀 Conteúdos abordados

* Modularização
* Reutilização de funções
* Formatação monetária
* Funções de alto nível
* Organização de código

## 🧠 Objetivo

Desenvolver a capacidade de:

* Reutilizar funções já criadas
* Centralizar informações em uma única função
* Organizar melhor a saída de dados
* Produzir código mais limpo e profissional

## 🗂️ Estrutura

Os arquivos seguem o padrão:

```text
moeda.py
teste.py
```

## 💡 Observações

* Este exercício é uma evolução direta dos módulos anteriores
