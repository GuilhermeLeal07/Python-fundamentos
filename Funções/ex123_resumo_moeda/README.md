# 💰 Resumo Monetário

Este exercício expande o módulo moeda criado anteriormente, adicionando uma função capaz de gerar um resumo completo sobre um valor informado.

## 🚀 Conteúdos abordados

- Modularização
- Reutilização de funções
- Organização de código
- Formatação monetária
- Funções de alto nível

## 📂 Arquivos

- moeda.py
- teste.py

## 🧠 Objetivo

Centralizar informações monetárias em uma única função, tornando o código mais organizado e fácil de reutilizar.
