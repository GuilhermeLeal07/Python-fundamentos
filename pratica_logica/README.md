# 🧠 Prática de Lógica

Esta pasta contém exercícios autorais focados no desenvolvimento da lógica de programação e resolução de problemas.

Diferente dos exercícios guiados do curso, aqui o objetivo é:
- Pensar soluções de forma independente
- Melhorar a clareza e organização do código
- Simular pequenos problemas do mundo real

---

## 🎯 Objetivos

- Fixar os fundamentos da linguagem Python
- Desenvolver raciocínio lógico
- Evoluir na escrita de código limpo e legível
- Preparar base para projetos mais complexos
