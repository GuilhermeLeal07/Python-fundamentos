# 🧠 Condicionais e Lógica em Python

Este módulo marca minha evolução no aprendizado de estruturas condicionais em Python.

Aqui comecei a trabalhar com tomadas de decisão dentro do código, utilizando estruturas como `if`, `elif` e `else`, além de operadores relacionais e operadores lógicos.

---

## 📚 Conteúdos Praticados

- Estrutura `if` e `else`
- Estrutura `elif`
- Operadores relacionais:
  - `>`, `<`, `>=`, `<=`, `==`, `!=`
- Operadores lógicos:
  - `and`
  - `or`
  - `not`
- Comparações numéricas e textuais
- Validação de entrada do usuário
- Pequenos sistemas baseados em decisão

---

## 💻 Exercícios Desenvolvidos

Os exercícios incluem:

- Verificação de maioridade
- Cálculo e validação de notas (aprovado/reprovado)
- Classificação por faixa etária
- Sistemas simples de decisão
- Mini desafios envolvendo múltiplas condições

---

## 🚧 Dificuldades Enfrentadas

Durante esse módulo enfrentei alguns desafios:

- Erros de **indentação**, que causavam falhas na execução
- Confusão inicial entre `=` e `==`
- Lógica invertida em algumas condições
- Esquecimento de converter dados com `int()` ou `float()`
- Uso incorreto de operadores lógicos, principalmente `and` e `or`

Esses erros foram fundamentais para meu aprendizado, pois me ajudaram a entender melhor como o Python interpreta cada condição.

---

## 📈 Evolução Técnica

Neste módulo comecei a desenvolver:

- Raciocínio lógico mais estruturado
- Organização mental para construir condições encadeadas
- Melhor leitura de código
- Atenção à identação e estrutura

Percebi que lógica de programação exige prática constante, e esse módulo foi essencial para fortalecer minha base.

---

## 🎯 Objetivo

Consolidar o entendimento sobre estruturas condicionais antes de avançar para estruturas de repetição (`while` e `for`).
